4.3 실습 5: 웹페이지 만들기

🌐 시나리오: 간단한 매물 소개 웹페이지

부동산 매물을 소개하는 웹페이지를 만들어주세요.

[구성]
1. 상단: 회사 로고, 메뉴
2. 메인: 매물 카드 3개
   - 사진 영역 (placeholder)
   - 주소, 가격, 면적
   - "자세히 보기" 버튼
3. 하단: 연락처

[디자인]
- 깔끔하고 모던한 스타일
- 모바일에서도 잘 보이게
- 네이비 + 화이트 색상

[기술]
- 단일 HTML 파일
- CSS, JS 모두 포함
- 바로 실행 가능하게

파일명: property_showcase.html

실습 진행:

  1. HTML 생성
  2. 브라우저에서 확인
  3. 수정 요청해보기
    • "색상 좀 더 밝게"
    • "카드 크기 키워줘"

5.1 실습 6: 이메일 자동 정리

Gmail에서 최근 1주일 메일을 분석해주세요.

[작업]
1. 중요도별로 분류:
   - 긴급 (당일 답변 필요)
   - 중요 (이번 주 안에)
   - 일반

2. 각 메일별로:
   - 발신자
   - 제목
   - 핵심 내용 한 줄 요약
   - 필요한 조치

[출력]
표로 정리해서 보여주세요.

5.2 실습 7: Drive 파일 정리

Google Drive에서 "계약서" 관련 파일을 찾아주세요.

[작업]
1. 모든 계약서 파일 검색
2. 날짜순으로 정리
3. 각 파일의:
   - 제목
   - 작성일
   - 주요 내용 요약

[추가]
파일명이 중구난방이면 정리 제안도 해주세요.

5.3 실습 8: 일정 자동 관리

Google Calendar를 확인해서:

[오늘 일정]
1. 오늘 예정된 미팅 리스트
2. 각 미팅의:
   - 시간
   - 장소
   - 참석자
   - 준비사항

[제안]
- 이동 시간 체크
- 일정 사이 여유시간 확인
