🤔 HTML이란?

HTML (HyperText Markup Language)

웹 페이지의 구조를 만드는 언어입니다.

🏗️ 집으로 비유하면:

  • HTML = 집의 뼈대 (기둥, 벽, 방)
  • CSS = 인테리어 (색상, 장식, 가구 배치)
  • JavaScript = 전기, 수도 (움직임, 동작)
💡 Tip: HTML은 프로그래밍 언어가 아닌 마크업 언어입니다!

📝 HTML 기본 구조

<!DOCTYPE html> <html lang="ko"> <head> <meta charset="UTF-8"> <title>페이지 제목</title> </head> <body> <h1>안녕하세요!</h1> <p>첫 번째 웹 페이지입니다.</p> </body> </html>

각 부분 설명:

  • <!DOCTYPE html> - HTML5 문서라는 선언
  • <html> - 전체 문서의 시작과 끝
  • <head> - 페이지 정보 (보이지 않음)
  • <body> - 실제로 보이는 내용

🏷️ 태그(Tag) 이해하기

태그는 꺾쇠괄호(< >)로 감싸진 명령어입니다.

기본 형태:

<태그이름>내용</태그이름>

예시: <h1>제목</h1>

닫는 태그가 없는 경우:

<br> (줄바꿈) <img src="이미지.jpg"> (이미지) <hr> (구분선)
💡 중요: 태그는 반드시 열었으면 닫아야 합니다!

✨ 자주 쓰는 태그 - 텍스트

태그 설명 예시
<h1> ~ <h6> 제목 (1이 가장 큼) <h1>큰 제목</h1>
<p> 문단 <p>단락입니다</p>
<br> 줄바꿈 첫줄<br>둘째줄
<strong> 굵은 글씨 (중요) <strong>중요!</strong>
<input> 한줄 입력 <input type="text">
<textarea> 긴 입력 <textarea></textarea>

✨ 자주 쓰는 태그 - 링크와 이미지

🔗 링크 만들기 (<a>)

<a href="https://www.naver.com">네이버로 가기</a> <a href="https://www.google.com" target="_blank">구글(새창)</a>
  • href - 이동할 주소
  • target="_blank" - 새 창에서 열기

🖼️ 이미지 넣기 (<img>)

<img src="cat.jpg" alt="고양이 사진"> <img src="cat.jpg" alt="고양이" width="300">
  • src - 이미지 파일 경로
  • alt - 설명 텍스트 (필수!)
  • width/height - 크기 조절

📋 목록 만들기

순서 없는 목록 (ul)

<ul> <li>사과</li> <li>바나나</li> <li>오렌지</li> </ul>
결과:
  • 사과
  • 바나나
  • 오렌지

순서 있는 목록 (ol)

<ol> <li>HTML 배우기</li> <li>CSS 배우기</li> <li>프로젝트 만들기</li> </ol>
결과:
  1. HTML 배우기
  2. CSS 배우기
  3. 프로젝트 만들기

📦 묶음 태그 (컨테이너)

<div> - 블록 영역

여러 요소를 그룹으로 묶을 때 사용 (한 줄 전체 차지)

<div> <h2>섹션 제목</h2> <p>섹션 내용</p> </div>

<span> - 인라인 영역

텍스트 일부분만 묶을 때 사용 (내용만큼만 차지)

<p>이것은 <span style="color: red;">빨간 글씨</span>입니다.</p>
💡 차이점: div는 줄바꿈O, span은 줄바꿈X

👨‍💻 실습 1: 첫 HTML 페이지 만들기

🎯 목표: 자기소개 페이지 기본 구조 만들기

포함할 내용:

  1. 큰 제목으로 자신의 이름
  2. 소제목으로 "자기소개"
  3. 자기소개 문단 2~3개
  4. 좋아하는 것 목록 (3개 이상)
  5. 좋아하는 웹사이트 링크 1개
<!DOCTYPE html> <html lang="ko"> <head> <meta charset="UTF-8"> <title>홍길동 소개</title> </head> <body> <h1>홍길동</h1> <h2>자기소개</h2> <p>안녕하세요! 저는 웹 개발을 공부하고 있습니다.</p> <h3>좋아하는 것</h3> <ul> <li>코딩</li> <li>음악 감상</li> <li>여행</li> </ul> <a href="https://www.github.com">GitHub</a> </body> </html>

🎨 CSS란?

CSS (Cascading Style Sheets)

HTML로 만든 구조를 예쁘게 꾸미는 언어입니다.

CSS로 할 수 있는 것:

  • ✅ 글자 색상, 크기, 폰트 변경
  • ✅ 배경색, 테두리 추가
  • ✅ 요소의 위치와 크기 조절
  • ✅ 애니메이션 효과

Before (HTML만): 밋밋한 검은 글씨

After (HTML + CSS): 화려한 디자인! 🎉

📍 CSS 적용하는 3가지 방법

1️⃣ 인라인 방식 (태그에 직접)

<h1 style="color: blue; font-size: 30px;">파란 제목</h1>

👉 간단하지만 관리가 어려워요

2️⃣ 내부 스타일 (head 안에)

<head> <style> h1 { color: blue; } p { font-size: 16px; } </style> </head>

👉 한 페이지에만 적용됩니다

3️⃣ 외부 파일 (추천! ⭐)

<head> <link rel="stylesheet" href="style.css"> </head>

👉 여러 페이지에서 공유 가능!

📖 CSS 기본 문법

선택자 { 속성: 값; 속성: 값; }

실제 예시:

h1 { color: blue; font-size: 32px; text-align: center; } p { color: #333; line-height: 1.6; }

🎯 구성 요소:

  • 선택자 - 어떤 요소를 꾸밀지 선택
  • 속성 - 무엇을 바꿀지
  • 값 - 어떻게 바꿀지

🎯 CSS 선택자

1. 태그 선택자

h1 { color: red; } p { font-size: 16px; }

→ 모든 h1, p 태그에 적용

2. 클래스 선택자 (.)

<p class="intro">소개글</p> .intro { color: blue; font-weight: bold; }

→ class="intro"인 요소에만 적용

3. 아이디 선택자 (#)

<div id="header">헤더</div> #header { background: gray; }

→ id="header"인 요소에만 적용 (한 페이지에 하나만!)

🎨 자주 쓰는 CSS 속성 - 텍스트

속성 설명 예시
color 글자 색 color: red;
font-size 글자 크기 font-size: 20px;
font-weight 글자 굵기 font-weight: bold;
text-align 정렬 text-align: center;
line-height 줄 간격 line-height: 1.5;
💡 색상 표현: red, #FF0000, rgb(255,0,0) 모두 빨강!

🎨 자주 쓰는 CSS 속성 - 박스

속성 설명 예시
background-color 배경색 background-color: yellow;
width 너비 width: 300px;
height 높이 height: 200px;
padding 안쪽 여백 padding: 20px;
margin 바깥 여백 margin: 10px;
border 테두리 border: 1px solid black;

📦 박스 모델 이해하기

모든 HTML 요소는 박스로 구성됩니다!

내용 (Content)
⬆️ padding (안쪽 여백)
⬆️ border (테두리)

⬆️ margin (바깥 여백)

div { width: 200px; /* 내용 너비 */ padding: 20px; /* 안쪽 여백 */ border: 2px solid black; /* 테두리 */ margin: 10px; /* 바깥 여백 */ }
💡 실제 요소 크기 = width + padding + border

🖱️ 마우스 반응 효과 (가상 클래스)

사용자의 동작에 따라 스타일을 변경할 수 있어요!

:hover - 마우스를 올렸을 때

a:hover { color: red; font-weight: bold; } button:hover { background-color: blue; }
예시: 마우스를 올려보세요!

:active - 클릭하는 순간

button:active { background-color: darkblue; transform: scale(0.95); }

👉 버튼을 누르는 순간만 적용됩니다

:focus - 입력칸을 선택했을 때

input:focus { border: 2px solid blue; outline: none; }

👉 주로 input, textarea 등에 사용

💡 조합 가능: a:hover, button:active 등으로 특정 태그에만 적용!

👨‍💻 실습 2: CSS로 꾸미기

🎯 목표: 자기소개 페이지에 스타일 적용하기

적용할 스타일:

  1. 제목(h1)을 파란색으로, 가운데 정렬
  2. 문단(p)의 글자 크기를 18px로
  3. 전체 배경색을 연한 회색으로
  4. 링크에 마우스를 올리면 색상 변경 (:hover 사용!)
<!DOCTYPE html> <html lang="ko"> <head> <meta charset="UTF-8"> <title>홍길동 소개</title> <style> body { background-color: #f5f5f5; padding: 20px; } h1 { color: #0066cc; text-align: center; } p { font-size: 18px; line-height: 1.6; } a { color: #0066cc; text-decoration: none; } a:hover { color: #ff6600; } </style> </head> <body> <!-- HTML 내용은 이전과 동일 --> </body> </html>

🎯 최종 프로젝트: 멋진 자기소개 페이지

✨ 만들 것:

  • 헤더 영역 (이름, 직업/학교)
  • 자기소개 섹션
  • 취미/관심사 목록
  • 연락처/소셜 링크

🎨 포함할 스타일:

  • 배경색과 글자색 조화롭게
  • padding, margin으로 여백 조절
  • 테두리나 그림자 효과
  • 호버 효과
💡 추가 도전: 이미지 추가, 목록 스타일 변경, border-radius로 둥근 모서리 만들기!
